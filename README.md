#  Bloom9

Bloom9 is a maternal health monitoring mobile application built with Flutter. It helps pregnant women monitor important health metrics, receive timely alerts, and stay informed throughout their pregnancy.

##  Features

*  Secure user authentication
*  Heart rate monitoring through a connected smartwatch
*  Real-time health dashboard
*  Push notifications and health alerts
*  Health data visualization
*  Cloud data storage using Supabase
*  AI-powered pregnancy risk assessment (in development)

##  Tech Stack

### Frontend

* Flutter
* Dart

### Backend

* Supabase
* Firebase Cloud Messaging (FCM)

### Machine Learning

* Python
* Scikit-learn

## Getting Started

### Prerequisites

* Flutter SDK
* Dart SDK
* Android Studio or VS Code
* Git

### Installation

1. Clone the repository

```bash
git clone https://github.com/<your-username>/Bloom9.git
```

2. Navigate to the project

```bash
cd Bloom9
```

3. Install dependencies

```bash
flutter pub get
```

4. Configure your Firebase and Supabase credentials.

5. Run the application

```bash
flutter run
```
